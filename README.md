# Projeto com .net 7 e Kafka

Proposta de treinamento para Avanade Brasil

# Comandos Docker

> docker-compose -f ./docker/ambiente-local.yml up -d --build
> docker-compose -f ./docker/ambiente-local.yml down --remove-orphans

> docker system prune
